# 项目进展控制文档

## 1. 最终产品目标

交付一个完整、可运行、模块化、可扩展的智能客服平台，覆盖：

- 文本客服
- 语音客服
- RTC 实时语音通话
- RAG 知识增强
- 业务系统联动
- AI 与人工协同
- 多渠道接入
- 多提供商适配
- 运营管理
- 日志、监控与测试

## 2. 当前总体目标

从零搭建参考实现，先完成文档与骨架，再逐模块开发并完成联调、测试、交付文档。

## 9. 风险项与阻塞项

- 默认本地 ASR 仅支持开发环境文本载荷，真实音频识别需切换正式 ASR 提供商
- 默认本地 TTS 为链路验证音频输出，生产环境建议切换正式 TTS 提供商
- OpenAI、Qdrant 等外部提供商适配器代码已交付，但当前未配置真实密钥与服务地址，联调当前不可验证
- HTTP 业务适配器已交付，但当前未接入外部业务系统，联调当前不可验证
